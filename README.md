# Ideabase
[Ideabase](https://ideabase.chat) is an open source Firebase alternative in China. We're building the features of Firebase using enterprise-grade open source tools  and the Rust programming language, it offers Firebase-like capabilities with robust security and top-tier performance.
- [x] Hosted MySQL Database.
  - [x] REST
  - [ ] Cache
  - [ ] Realtime subscriptions
- [ ] Authentication + Authorization
- [ ] AI + RAG
- [ ] AI + MCP
- [ ] File Storage
- [ ] Dashboard
